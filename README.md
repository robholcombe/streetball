# Project README #

### Prerequisites ###

* Git + LFS [help here](http://en.joysword.com/posts/2016/03/setting_up_github_for_unity_projects/)
* Unity3D
* Visual Studio Community or VS Code
* Visual Studio Tools for Unity


### Getting Started ###

1. Install all prerequisites in order above
1. Git clone repo (only after LFS is setup)
1. Open Visual Studio Community (VSC) and setup Unity Tools extension
1. Open Unity project and set VSC as default editor


### Documentation and Workflow ###
* Workflow and tasks are managed through a [Trello](https://trello.com/) kanban board
* Design documentation is managed through [Google Drive](https://drive.google.com)
* Development documentation is managed through the [Streetball Project Wiki](https://bitbucket.org/hulkamaphone/streetball/wiki/)
